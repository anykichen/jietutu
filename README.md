# 📸 SnapFloat 截图工具

> 功能强大的 Windows 截图工具，支持悬浮窗口、托盘常驻、开机自启动

---

## ✨ 功能特性

| 功能 | 说明 |
|------|------|
| ✂️ 局部截图 | 拖动鼠标框选任意区域进行截图 |
| 🖥️ 全屏截图 | 一键截取整个屏幕 |
| 📌 悬浮显示 | 截图窗口始终显示在所有程序之上 |
| 📋 复制截图 | 一键将截图复制到剪贴板 |
| 📝 文字复制 | 截图内容文字可查看和复制 |
| 💾 保存图片 | 支持另存为 PNG / JPG 格式 |
| 🔒 托盘常驻 | 点关闭只隐藏，不会真正退出 |
| 🚀 开机自启 | 首次运行自动设置开机启动 |
| 🔁 单实例 | 同时只能运行一个，防止重复打开 |
| ⌨️ 全局热键 | 系统级快捷键，任意界面可触发 |

---

## 🚀 快速开始

### 方法一：直接运行（需要 Python）

1. 安装依赖：
   ```
   pip install PyQt5 pywin32 Pillow
   ```

2. 运行程序：
   ```
   python snapfloat.py
   ```

### 方法二：打包为独立 EXE（推荐）

双击运行 `打包为EXE.bat`，等待约 1-2 分钟，完成后在 `dist` 目录找到 `SnapFloat截图工具.exe`。

---

## ⌨️ 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + Alt + A` | 局部截图（框选区域） |
| `Ctrl + Alt + F` | 全屏截图 |
| `ESC` | 取消截图选择 |
| `Enter` | 确认截图选区 |
| `Ctrl + C`（截图窗口内） | 复制截图到剪贴板 |

---

## 🖱️ 托盘图标操作

| 操作 | 功能 |
|------|------|
| 左键单击 | 弹出快捷操作菜单 |
| 右键单击 | 弹出完整功能菜单 |
| 右键 → 退出程序 | 真正退出软件 |

---

## 📋 截图窗口操作

| 按钮 | 功能 |
|------|------|
| 📋 | 复制截图到剪贴板 |
| 📝 | 查看文字 / 手动复制 |
| 💾 | 保存截图到文件 |
| 📌 | 切换窗口置顶状态 |
| ✕ | 关闭截图（不退出程序） |

---

## 🔧 系统要求

- **操作系统**：Windows 10 / 11（64位）
- **Python 版本**：3.8 或更高（仅源码运行需要）
- **依赖库**：PyQt5、pywin32、Pillow
